# 🖥️ 8085 Microprocessor Simulator

A simple and interactive **8085 Microprocessor Simulator** that allows users to write, execute, and visualize 8085 assembly programs step-by-step.

---

## 🚀 Features

- ✍️ Write 8085 Assembly Code
- ▶️ Execute Instructions Step-by-Step
- 🔄 View Register & Memory State in Real-Time
- 🧠 Supports Common 8085 Instructions
- 📊 User-Friendly Interface
- 💾 Load & Save Programs (optional)
